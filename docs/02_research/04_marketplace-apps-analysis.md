## ✅ УСТАНОВИТЬ — Фаза 1

### 1. ZikPro Invoice OCR
> **Для:** отдела снабжения и бухгалтерии

- AI-распознавание счетов-фактур из PDF и фото
- **Мобильная камера** — снабженец фотографирует бумажный счёт прямо на площадке
- Автоматическое создание Purchase Invoice в ERPNext
- Распознавание и сопоставление поставщиков
- Защита от дублирования (не введёшь один счёт дважды)
- Валидация сумм и налогов
- Использует DeepInfra API (не OpenAI) — дешевле, без отправки данных в ChatGPT
- **v15 и v16** ✅

> [!TIP]
> Почему не IPCONNEX AI Invoice? ZikPro превосходит по всем параметрам: мобильная камера, сопоставление поставщиков, защита от дублей, валидация, v16. IPCONNEX использует ChatGPT — данные счетов уходят во внешний сервис.

**GitHub:** Marketplace / ZikPro (официальный партнёр Frappe)

---

### 2. Mint — Банковская сверка
> **Для:** бухгалтерии

- Продвинутая сверка банковских выписок (React UI на `/mint`)
- Smart-matching по сумме, дате, референсу
- Массовая сверка (bulk reconciliation)
- Разделение транзакций (один платёж — несколько счетов)
- Движок правил для автоклассификации повторяющихся платежей
- Обнаружение межбанковских переводов
- Отмена действий (undo)

> [!IMPORTANT]
> Строительная компания обрабатывает сотни банковских транзакций ежемесячно (оплата материалов, субподрядчикам, приём оплаты по КС). Smart-matching кратно ускоряет сверку.

**GitHub:** https://github.com/The-Commit-Company/mint  
**Версия:** v15 (проверить v16 в `pyproject.toml` репозитория)

---

### 3. ClefinCode Chat — Омниканальный мессенджер
> **Для:** всех — особенно прорабов и начальников участков

Каналы:
- ✅ **WhatsApp** (через Twilio)
- ✅ **Telegram**
- ✅ **Facebook Messenger**
- ✅ **Instagram DM**
- ✅ Внутренний чат ERPNext

Ключевые преимущества:
- Привязка разговоров к DocType (Проект, Заказ на закупку и т.д.)
- История коммуникаций сохраняется даже при увольнении сотрудника
- Мобильные приложения iOS/Android
- **v16** ✅ (PR на миграцию смерджен в марте 2026)

> [!NOTE]
> WhatsApp-интеграция требует аккаунт Twilio (платный). Telegram работает бесплатно. Прорабы в Казахстане активно используют WhatsApp и Telegram — объединить всё в ERPNext очень ценно.

**GitHub:** https://github.com/clefincode/clefincode_chat  
**Цена:** Бесплатно (open-source)

## ⚠️ ТЕСТИРОВАТЬ — на staging перед установкой

### 4. SaaS Theme — Современный UI
> **Для:** всех пользователей

- Двойной сайдбар с быстрым переключением модулей
- Переработанные формы с крупными полями ввода
- Слайд-редактор для дочерних таблиц (вместо мучений с inline-таблицами)
- Улучшенные вложения с иконками типов файлов
- Тёмный режим
- Современная двухколоночная страница логина

> [!WARNING]
> Разработчик — индивидуальный (vineyrawat). Долгосрочная поддержка не гарантирована. Тестировать тщательно.

**GitHub:** https://github.com/vineyrawat/saas_theme  
**Версия:** v16 ✅

---

### 5. WorkBoard — Командные задачи
> **Для:** начальников участков, прорабов

- Назначение задач конкретным сотрудникам
- **Повторяющиеся задачи** (ежедневные/еженедельные/ежемесячные)
- **Триггеры событий** — авто-создание задачи при событиях ERPNext
- Чеклисты для проверки выполнения
- Контроль своевременности (вовремя / с опозданием)
- Двухшаговый workflow: «Выполнено» (исполнитель) → «Завершено» (руководитель)

> [!TIP]
> Повторяющиеся задачи полезны для рутинных проверок (ежедневный обход безопасности, еженедельные отчёты). Done → Completed — хорошо ложится на схему прораб → начальник участка.

**GitHub:** https://github.com/bhavesh95863/workboard  
**Версия:** v16 ✅

---

### 6. Paneview — Split-screen навигация
> **Для:** всех — ускоряет работу с большими списками

- Показывает список записей слева, форму редактирования справа
- Мгновенное переключение без перезагрузки страницы
- Регулируемое соотношение панелей

> [!WARNING]
> Закрытый код. Распространяется только через Marketplace. v16 не подтверждён официально (Kanak указывает v15/v16, но нужна проверка).

### 7. Laborers Management — Подённые рабочие
> **Для:** начальников участков

- Управление дневными/временными рабочими
- Распределение по проектам и ЦЗ
- Учёт посещаемости, переработок, удержаний
- Импорт данных (мастера, смены, супервайзеры)
- Автоматическое формирование расчётных листов

> [!WARNING]
> v16 не подтверждён. Репозиторий закрытый. Это **учёт рабочей силы и зарплаты**, а НЕ ежедневные отчёты о ходе работ. Для отчётов прораба по прогрессу стройки нужны кастомные DocType.

## ❌ ПРОПУСТИТЬ — с обоснованием

| Приложение | Почему пропускаем |
|-----------|-------------------|
| **Advanced Procurement** | Приложение не найдено / ориентировано на розницу. Встроенный модуль Buying (Material Requests, RFQ, Supplier Quotation) достаточен. |
| **IPCONNEX AI Invoice** | ZikPro лучше по всем параметрам. IPCONNEX отправляет данные в ChatGPT. |
| **Cargo Management** | Для грузовых экспедиторов (Import/Export), не для собственного автопарка. Используйте встроенный Fleet Management + Delivery Trip. |
| **ClickPost** | eCommerce-доставка через FedEx/DHL. Не для стройки. |
| **RuknTheme** | Только v13/v14. Мёртвый проект. |
| **Productivity Next** | Это ПО для **слежки** за сотрудниками (кейлоггер, мониторинг сайтов), а НЕ аналитика ERP. v14/v15. |
| **Report Builder** | Уже встроен в Frappe. Для продвинутой аналитики — Frappe Insights. |
| **EAE** | Узкий HR-модуль (авансы), v16 не подтверждён. |
| **Document Progress Tracker** | v15 only, платный. Для согласования АВР используйте **встроенный Frappe Workflow** — бесплатно, v16, лучше подходит для цепочек утверждения. |
| **QuickDo** | Личные to-do. ERPNext Projects + Tasks + Assignments покрывают задачи стройки значительно лучше. |

## Встроенные альтернативы (не требуют установки)

| Потребность | Встроенное решение |
|------------|-------------------|
| Согласование АВР | **Frappe Workflow** — кастомные состояния, роли, email-уведомления |
