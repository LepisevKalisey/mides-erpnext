## Контекст

В процессе разработки директорского дашборда обнаружилась критическая проблема: шрифты в реализованном интерфейсе выглядели «мыльно» и дёшево по сравнению с эталонным `SharedDesign.html`.

Первоначально в проекте использовались:
- **Inter** (тело UI) — хорошая кириллица, но рендерится мягче
- **Plus Jakarta Sans** (бренд) — нет кириллицы
- **JetBrains Mono** (числа) — тёмный стиль, не соответствует SharedDesign
- CSS токены через `oklch()` — абстракции, отличающиеся от точных hex

Дополнительно: использование `oklch()` в CSS-переменных при отображении в браузере приводило к незначительным расхождениям цветового рендеринга, особенно в режимах высокой контрастности.

## Решение

Принять шрифтовую систему из `SharedDesign.html` как единый источник истины:

| Роль | Шрифт | Применение |
|------|-------|------------|
| **Основной UI** | **Fira Sans** 300/400/500/600/700 | Весь текст интерфейса, таблицы, формы |
| **Заголовки / числа** | **Fira Code** 400/500/600/700 | h1, h2, h3, KPI значения, числа в таблицах |

Цветовые токены переведены с `oklch()` на точные hex-значения из SharedDesign:

```css
--c-primary:   #181d26  /* тёмно-синий (кнопки, сайдбар) */
--c-body:      #334155  /* основной текст */
--c-muted:     #475569  /* вспомогательный текст */
--c-hairline:  #dddddd  /* рамки */
--c-danger:    #b42318  /* ошибки, отрицательные числа */
--c-warn:      #b54708  /* предупреждения */
--c-success:   #067647  /* успех, положительные числа */
```

## Альтернативы

| Вариант | Причина отказа |
|---------|---------------|
| Остаться на Inter | Иной визуальный характер, не совпадает с SharedDesign |
| Geist (Vercel) | Нет истории в проекте, нет обоснования |
| Manrope / Onest | Нет кириллицы с нужным весом |

## Последствия

- Весь интерфейс теперь использует единую шрифтовую систему из SharedDesign.html
- `globals.css` заменён: убраны `oklch()` абстракции, все токены — точные hex
- `layout.tsx` заменён: Inter/Jakarta/JetBrains Mono → Fira Sans + Fira Code
- Числовые ячейки таблиц (`fp-num-*`) используют Fira Code с `tabular-nums`
- Заголовки (`h1`, `h2`, `h3`, `.page-title`, `.font-brand`) — Fira Code
